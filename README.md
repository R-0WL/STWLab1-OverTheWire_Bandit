------Walktrough of  Bandit levels by Over The Wire :D------
- In the 'bandit-report.md' file you will find all the documentation for each level.
- Each level has its real-time terminal recording in the 'recsBanditLevels' directory.
-- To play it is necessary to have the asciinema command followed by the play parameter and the file to run.
```
sudo apt install asciinema
asciinema play <file>
```
-Passwords for each level are stored in the 'passwords' directory.
